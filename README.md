# Outgunned
Unofficial Outgunned RPG system for Foundry VTT

# ALPHA Version

## ROLL MACRO:
``` game.outgunned.OutgunnedRollerDialog.createDialog({rollName:"CUSTOM ROLL", total:0, modifier:0}) ```


## ACTOR SHEET:

### ROLL
Select Attribute and Skill, add modifier or gamble, click ROLL button on the bottom of the skill.

### ATTRIBUTES AND SKILLS:
Hold *SHIFT* to increase, hold *SHIFT+ALT* do decrease

Any other "tracker" should be left mouse increase right mouse to decrease.
